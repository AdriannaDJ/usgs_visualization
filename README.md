# usgs_visualization
Creating a map visualization using Leaflet to show the magnitude and depth of all earthquakes in the last 7 days.

## Outputs
- Produce an interactive world map
- Include all data points provided from data source
- Each earthquake's circle marker radius is based on magnitude and color based on depth
- Each earthquake marker is clickable with a popup of selective earthquake data

## Resources
- Referenced lesson activities from Module 15 Mapping.
- Data Source: USGS https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/all_week.geojson
- Tile Source: Open Street Map https://www.openstreetmap.org/copyright
- Code for converting timestamp into readable date/time: https://sentry.io/answers/convert-unix-timestamp-to-date-and-time-in-javascript/
